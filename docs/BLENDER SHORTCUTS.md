# BLENDER SHORTCUTS

## View Navigation

- Rotate view: middle mouse drag
- Pan view: Shift + middle mouse drag
- Zoom view: mouse wheel, or Ctrl + middle mouse drag
- Focus selected object: Numpad .
- Frame all objects: Home
- Front view: Numpad 1
- Right view: Numpad 3
- Top view: Numpad 7
- Opposite view: Ctrl + Numpad 1/3/7
- Camera view: Numpad 0
- Toggle perspective/orthographic: Numpad 5

If you do not have a numpad, enable:
Edit > Preferences > Input > Emulate Numpad

## Selection

- Select object: left click
- Select multiple: Shift + left click
- Select all: A
- Deselect all: Alt + A
- Box select: B
- Circle select: C
- Invert selection: Ctrl + I

## Moving Around Large Maps

- Focus on selected: Numpad .
- Walk/fly navigation: Shift + `
- Increase/decrease walk speed: mouse wheel while in walk mode
- Exit walk/fly mode: Esc or right click
- Confirm walk/fly position: left click

## Transform Tools

- Move: G
- Rotate: R
- Scale: S
- Confirm transform: left click or Enter
- Cancel transform: right click or Esc
- Move on X axis: G, then X
- Move on Y axis: G, then Y
- Move on Z axis: G, then Z
- Rotate around X/Y/Z: R, then X/Y/Z
- Scale on X/Y/Z: S, then X/Y/Z
- Type exact value: press number after axis, example G, X, 10

## Common Editing

- Undo: Ctrl + Z
- Redo: Shift + Ctrl + Z
- Duplicate: Shift + D
- Delete selected: X
- Hide selected: H
- Hide unselected: Shift + H
- Unhide all: Alt + H

## Modes

- Object/Edit mode toggle: Tab
- Vertex select: 1 in Edit Mode
- Edge select: 2 in Edit Mode
- Face select: 3 in Edit Mode
- Search command menu: F3

## View Display

- Solid view: Z, then 2
- Material Preview: Z, then 3
- Rendered view: Z, then 4
- Wireframe: Z, then 1
- Toggle X-Ray: Alt + Z

## Useful For Imported Perfect Dark Maps

- Use Material Preview to see textures.
- Use Numpad . often after selecting a room, prop, tile, or waypoint.
- Large imported maps contain many objects, so hiding selected/unselected objects is useful.
- Tiles, rooms, props, portals, and waypoints are usually organized into separate collections in the Outliner.
